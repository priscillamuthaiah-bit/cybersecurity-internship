TOOL FAMILIARIZATION



Wireshark



Wireshark is used to capture and analyze network packets.

It helped me understand how data flows across a network.



Nmap



Nmap scans networks to find open ports and running services.

I used it to identify possible entry points in a target system.



Burp Suite



Burp Suite is a web proxy tool used to inspect and modify HTTP requests.

It helps in testing web application security.



Netcat



Netcat is a networking utility for reading and writing data over network connections.

It is useful for debugging, banner grabbing, and basic client-server testing.
